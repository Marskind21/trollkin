# Trollkin V3.2

## Changelog:
  - Added Dried Ghast to the Ghast category. (It apperently wasn't in the pool before?)
  - Added Beetroot Stew to the Just Annoing category.
  - Added Fire Charge to the Blaze category.
  - Added new category Trollkin which contains two items: Trollkin and Lit Trollkin.
  - You are now no longer forced to ride firework rockets which could previously lead to an unexpected death.
  - Players and most mobs change their size depending on how much health they have.
  - The pack now uses my Item Pool Changer extension instead of its own implementation for switching the item pool.
  - All Trollkin features (except advancements, recipes and resoucepack features) are disabled when an item pool that doesn't contain the `trollkin:default` item pool tag is selected. 
  - Recipes are now also given on game start and not only when the data pack is reloaded.
